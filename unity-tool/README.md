# 🧠 SynapseGraph (Unity Tool)

Package này cung cấp công cụ phân tích kiến trúc mã nguồn sử dụng **Roslyn AST (Abstract Syntax Tree)**.

## ✨ Tính năng
* **Deep Analysis**: Soi sâu logic hàm, bỏ qua comment và mã rác.
* **Drag & Drop UI**: Kéo thả thư mục quét trực tiếp trên Editor.
* **JSON Export**: Xuất dữ liệu chuẩn hóa cho công cụ visualizer.

## 🚀 Hướng dẫn nhanh
1. Truy cập: **Tools > SynapseGraph > Export Architecture**.
2. Kéo thả thư mục Script cần quét vào danh sách.
3. Bấm **SCAN & EXPORT**.
