# 🧠 SynapseGraph Ecosystem

Hệ sinh thái phân tích kiến trúc mã nguồn Unity mạnh mẽ, sử dụng công nghệ **Roslyn AST** để trích xuất sơ đồ liên kết giữa các lớp một cách chính xác tuyệt đối.

## 📁 Thành phần dự án
* **unity-tool**: Unity Package (UPM) giúp quét mã nguồn và xuất JSON.
* **web-visualizer**: Công cụ Web kéo thả file JSON để hiển thị sơ đồ Neural Network trực quan.

## 🚀 Cách cài đặt Unity Tool
Trong Unity Package Manager, chọn **Add package from git URL** và dán:
`https://github.com/huuthao198/SynapseGraph.git?path=/unity-tool`

## 🌐 Trình xem trực quan (Web)
Bản demo online: `https://huuthao198.github.io/SynapseGraph/`
